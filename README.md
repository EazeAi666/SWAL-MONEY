# 🚀 SWAL Money
**Empowering Nigerians with Seamless, Borderless, and Secure Finance.**

SWAL Money is a next-generation fintech platform designed for the Nigerian market. Built to provide the speed of Palmpay with the robustness of modern banking, SWAL Money bridges the gap between traditional saving and digital spending.

## 📱 Features
 * **Instant Transfers:** Send and receive money across all Nigerian banks with 99.9% uptime.
 * **Lifestyle Payments:** Pay for electricity, cable TV (DSTV/GOTV), and data bundles in seconds.
 * **Smart Savings:** Automated savings "vaults" with competitive interest rates to beat inflation.
 * **Virtual Cards:** Generate USD/NGN virtual cards for international subscriptions and global shopping.
 * **Rewards System:** Earn points and cashback on every transaction—making money move for you.

## 🛠 Tech Stack
Our architecture is built for high concurrency and military-grade security:
 * **Frontend:** React / Vite (Web)
 * **Backend:** Node.js (TypeScript)
 * **Database:** PostgreSQL (Transactional data) & Redis (Caching/Speed)
 * **Infrastructure:** AWS (Auto-scaling)
 * **Security:** AES-256 Encryption, Biometric Auth, and PCI-DSS Compliance.

## 🚀 Getting Started
### Prerequisites
 * Node.js (v18+)

### Installation
 1. **Clone the repository:**
   ```bash
   git clone https://github.com/swal-org/swal-money.git
   cd swal-money
   ```
 2. **Install dependencies:**
   ```bash
   npm install
   ```
 3. **Environment Setup:**
   Create a .env file in the root directory and add your keys:
   ```env
   GEMINI_API_KEY=your_key
   ```
 4. **Run the application:**
   ```bash
   npm run dev
   ```
